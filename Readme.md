# Smart Blog Summarizer

A full-stack blog and article summarization platform built to transform long-form content into concise, customizable summaries through a clean modern dashboard interface.

## Overview

Smart Blog Summarizer allows users to:

- Paste blog posts, reports, or article content
- Upload supported files for summarization
- Customize summary length (Short / Medium / Long)
- Choose tone style (Simple / Professional)
- Select output format (Bullet Points / Paragraph)
- View results in a responsive dashboard
- Copy generated summaries instantly

This project was created as a learning-focused full-stack application to improve frontend UI/UX design, React architecture, and backend integration.
