# Smart Blog Summarizer

A modern React + Material UI web application that summarizes blogs, articles, and reports into concise outputs based on customizable preferences.

## Features

- Text or file-based input
- Summary length options (Short / Medium / Long)
- Tone customization (Simple / Professional)
- Output formats (Bullet Points / Paragraph)
- Responsive dashboard UI
- Scrollable summary output
- Copy-to-clipboard functionality

## Tech Stack

- React
- TypeScript
- Material UI (MUI)
- Formik

## UI Highlights

- Premium dashboard layout
- Sticky input panel
- Scrollable output section
- Modern card-based design

## Purpose

This project was built for learning and improving frontend design, component architecture, and user experience practices.
